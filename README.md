​أ### 3.تحديد الرقم (Even or Odd)
​الهدف من البرنامج: معرفة هل الرقم الذي يدخله المستخدم "زوجي" (يقبل القسمة على 2) أم "فردي".
​الأداة السحرية (Modulus %):
استخدمنا علامة % اللي بتحسب "باقي القسمة".
​لو n % 2 == 0 (الباقي صفر) -> الرقم زوجي.
​لو n % 2 != 0 (يوجد باقي) -> الرقم فردي.
​اللمسة الاحترافية في الـ Output:
بدل ما نطبع كلمة "زوجي" بس، خليت البرنامج يطبع الرقم نفسه اللي المستخدم دخله (cout << n << ...) عشان النتيجة تكون واضحة ومخصصة للمستخدم
