## 📁 محتويات المشروع:

### 1. برنامج حساب المضروب (Factorial)
* **الملف:** `Get_the_Factorial.cpp`
* **الفكرة:** البرنامج بيطلب من المستخدم رقم، وبيستخدم الـ `for loop` عشان يضرب الأرقام من 1 لحد الرقم ده.
* **اللي اتعلمته:** أهمية الـ Initialization (إني أبدأ المتغير `fact` بـ 1 مش 0 عشان الضرب يطلع صح).

### 2. برنامج جمع الأرقام (Sum)
* **الملف:** `Sum_of_2Number.cpp`
* **الفكرة:** برنامج بسيط لجمع رقمين وتدريب على مدخلات ومخرجات المستخدم (`cin`, `cout`).
